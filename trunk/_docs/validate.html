$Validate->print_text($field_name, $field_label, $attribute_array = array())

Generate a text input field.

$Validate->print_text('field_name', 'Field Label');

	<p class="field_name">Field Label</p>
	<div class="field_input">
		<input type="text" name="field_name" id="field_name" value="">
	</div>
</div>

Select Field

$Validate->print_select($field_name, $field_label, $field_value, $attribute_array = array())

Generate a select input field.

Single Value

Allow a single value to be selected (default).

$Validate->print_select('field_name', 'Field Label', array('1' => 'One', '2' => 'Two')); 
<div class="field_wrapper">
	<p class="field_name">Field Label</p>
	<div class="field_input">
		<select name="field_name" id="field_name">
			<option value="1">One</option>
			<option value="2">Two</option>
		</select>
	</div>
</div>

Note:  In single value mode, all data will be returned in string form.

Note:  $attribute_array is explained here.

Multiple Value

Passing the multiple attribute to $attribute_array will enable multiple selections. $Validate->print_select('field_name', 'Field Label', array('1' => 'One', '2' => 'Two'), array('multiple' => 'multiple')); 

<div class="field_wrapper">
	<p class="field_name">Field Label</p>
	<div class="field_input">
		<select name="field_name[]" id="field_name" multiple="multiple">
			<option value="1">One</option>
			<option value="2">Two</option>
		</select>
	</div>
</div>

Note:  In multiple value mode, all data will be returned in array form.

Note:  $attribute_array is explained here.

Hide Field Titles

$Validate->hide_titles($hide_once = FALSE)

Hide field titles. Optionally only hide the next element title. Returns a Validate object so method can be chained.

Hide All Field Titles

$Validate->hide_titles();

Hide Next Field Title

$Validate->hide_titles(TRUE);

Example:

$Validate->hide_titles(TRUE)->print_text('field_name', 'Field Label');
<div class="field_wrapper">
	<div class="field_input">
		<input type="text" name="field_name" id="field_name" value="">
	</div>
</div>

Show Field Titles

$Validate->show_titles()

Show field titles. Titles are enabled by default so this method will only need to be used if titles have been turned off elsewhere. Returns a Validate object so method can be chained.

$Validate->show_titles();

Hide Field Errors

$Validate->hide_errors($hide_once = FALSE)

Hide field errors. Optionally only hide the next element's errors. Returns a Validate object so method can be chained.

Hide All Field Errors

$Validate->hide_errors();

Hide Next Field Error

$Validate->hide_errors(TRUE);

Note:  Only error strings are hidden. All validation_error CSS classes will still be applied to the field.

Show Field Errors

$Validate->show_errors()

Show field errors. Errors are enabled by default so this method will only need to be used if errors have been turned off elsewhere. Returns a Validate object so method can be chained.

$Validate->show_errors();

Attribute Array

The attribute array allows custom attributes to be applied to form elements.

Single Attribute:

$Validate->print_text('field_name', 'Field Label', array('class' => 'blue')); 
<div class="field_wrapper">
	<p class="field_name">Field Label</p>
	<div class="field_input">
		<input type="text" name="field_name" id="field_name" value="" class="blue">
	</div>
</div>

Multiple Values For Single Attribute:

$Validate->print_text('field_name', 'Field Label', array('class' => array('blue', 'red'))); 
<div class="field_wrapper">
	<p class="field_name">Field Label</p>
	<div class="field_input">
		<input type="text" name="field_name" id="field_name" value="" class="blue red">
	</div>
</div>

Note:  For most attributes, it is simpler to combine into one value array('class' => 'blue red').

Multiple Attribute:

$Validate->print_text('field_name', 'Field Label', array('class' => 'blue', 'placeholder' => 'Placeholder Text')); 
<div class="field_wrapper">
	<p class="field_name">Field Label</p>
	<div class="field_input">
		<input type="text" name="field_name" id="field_name" value="" class="blue" placeholder="Placeholder Text">
	</div>
</div>

Required Field

reqd

Will require a non empty data value.

array('field_name' => array('reqd' => 'Field is required.'))

Conditional Field

cond

Passing an empty value will disable all following rules. In the case of a non empty value, all following rules will be applied.

array('field_name' => array('cond' => '', 'following_rule' => 'Rule error.'))

Note:  The conditional error string will never be displayed so there is no need to set one.

Match Field

match[field_name]

Will ensure the field value matches that of the specified field.

array('field_name' => array('match[field_name_2]' => 'field_name must match field_name_2'))

Password Confirmation Example

$rules = array(
	'password' => array(
		'reqd' => 'Password is required.'
		'min[8]' => 'Password must be at lease 8 characters.',
	),
	'password_confirm' => array(
		'reqd' => 'Confirmation is required.'
		'match[password]' => 'Invalid password confirmation.'
	)
)

Note:  There is no need to apply additional rules to password_confirm since it will inherit the match field rules.

Maximum Value Length

max[integer_length]

Checks that the value string length is less than or equal to integer_length.

array('field_name' => array('max[10]' => 'Value length is greater than 10 characters.'))

Minimum Value Length

min[integer_length]

Checks that the value string length is more than or equal to integer_length.

array('field_name' => array('min[10]' => 'Value length is less than 10 characters.'))

Exact Value Length

exact[integer_length]

Checks that the value string length is exactly integer_length.

array('field_name' => array('exact[10]' => 'Value length is not 10 characters.'))

Alpha Value

alpha

Checks that the value string is comprised of alpha characters.

array('field_name' => array('alpha' => 'Value is not an alpha string.'))

Note:  alpha will allow spaces. To prohibit spaces, the no_space rule must also be applied.

Alphanumeric Value

alphanumeric

Checks that the value string is comprised of alphanumeric characters.

array('field_name' => array('alphanumeric' => 'Value is not an alphanumeric string.'))

Note:  alphanumeric will allow spaces. To prohibit spaces, the no_space rule must also be applied.

No Spaces

no_space

Checks that the value string does not contain spaces.

array('field_name' => array('no_space' => 'Value cannot contain space characters.'))
