Controllers

Controller Basics

A controller is a class file that is named in a way that it can be associated with a URI.

For example, if you request:

mysite.com/blog/index/

The framework would attempt to load the controller class Blog_Controller, and call the function Blog_Controller:index().

The "index" function is loaded by default if the second segment of the URI is empty. The request above could be shortened to:

mysite.com/blog/

The controller class file for the example above would be located in

www/framework/controller/Blog.php

Multiple Word Controllers

Controllers can be made up of multiple words. In a sample request for:

mysite.com/blog_comment/list/

The framework would attempt to load the controller class Blog_Comment_Controller, and call the function Blog_Comment_Controller:list().

The controller class file for the example above would be located in

www/framework/controller/Blog_Comment.php

Naming Guidelines

  • Each word begins with a capital letter
  • Singular
